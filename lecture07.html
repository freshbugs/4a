§1.7: Linear independence

TL;DR The columns of a matrix are linearly independent if and only if echelon form has a pivot in every column. There are other equivalent definitions.

Review

Suppose $A$ is a non-augmented matrix with $m$ rows.

The columns of $A$ span $\mathbb{R}^m$ ⇔ echelon form has a pivot in every row.

The opposite of linearly independent is linearly dependent.

Linear independence (easy definition)

Suppose $A$ is a non-augmented matrix.

The columns of $A$ are linearly independent ⇔ echelon form has a pivot in every column.

Linear independence (equivalent definition on p57)

Suppose $\{ \mathbf{u}, \mathbf{v}, \mathbf{w} \}$ is a set of three vectors in $\mathbb{R}^m$. It could be any number of vectors, but let's do three to make it easy.

$\{ \mathbf{u}, \mathbf{v}, \mathbf{w} \}$
is linearly independent 		⇔ $x \mathbf{u} + y \mathbf{v} + z \mathbf{w} = \mathbf{0}$
has only the trivial solution.

Of course $x \mathbf{u} + y \mathbf{v} + z \mathbf{w} = \mathbf{0}$ always has at least one solution. If it has exactly one solution then they are linearly independent.

$\{ \mathbf{u}, \mathbf{v}, \mathbf{w} \}$
is linearly dependent 		⇔ $x \mathbf{u} + y \mathbf{v} + z \mathbf{w} = \mathbf{0}$
has a non-trivial solution.

A non-trivial solution is $a \mathbf{u} + b \mathbf{v} + c \mathbf{w} = \mathbf{0}$ where at least one of $a, b, c$ is non-zero. If there is a non-trivial solution then there are infinitely many.

Linear independence (equivalent definition on p58)

Suppose $A$ is a non-augmented matrix.

The columns of $A$ are linearly independent ⇔ $A \mathbf{x} = \mathbf{0}$ has only the trivial solution.

$A \mathbf{x} = \mathbf{0}$ is homogeneous, so it always has at least one solution. If it has exactly one solution then the columns are linearly independent.

The columns of $A$ are linearly dependent ⇔ $A \mathbf{x} = \mathbf{0}$ has infinitely many solutions.

If $A \mathbf{x} = \mathbf{0}$ has any non-trivial solution then it has infinitely many.

Linear independence (equivalent definition on p59)

$\{\mathbf{u}, \mathbf{v}\}$ is linearly dependent if and only if one of them is a multiple of the other.

This only applies if there are just two vectors.

Theorem 7

A set of vectors is linearly dependent ⇔ one of them is a linear combination of the others.

If one of the vectors is a multiple of another then they are linearly dependent, but that is not the only way.

Theorem 8

Any set of $n+1$ or more vectors in $\mathbb{R}^n$ is linearly dependent.

A random set of $n$ or fewer vectors in $\mathbb{R}^n$ is probably linearly independent, but not always.

Theorem 9

Any set of vectors that includes the zero vector is linearly dependent.
