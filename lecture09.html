Terminology

  • The $(i,j)$-entry of $A$ is in the $i$th row and $j$th column.
  • The main diagonal goes diagonally from the top left entry.
  • A diagonal matrix: the entries not on the main diagonal are all $0$.
  • The $n \times n$ identity matrix $I_n$: the entries on the main diagonal are all $1$, and the entries not on the main diagonal are all $0$.
  • The $m \times n$ zero matrix $0_{m,n}$: all of the entries are $0$.

+, -, ×

Add and subtract matrices in the obvious way. They have to be the same shape.

eg. $ \left[ \begin{array}{cc} 1 & 2 \\ 3 & 5 \end{array} \right] + \left[ \begin{array}{cc} 1 & 0 \\ 1 & -1 \\ 0 & 0 \end{array} \right]$ is undefined.

Multiply a scalar times a matrix in the obvious way.

Multiply a matrix times a matrix in a not-at-all-obvious way.

Suppose $A$ is an $m \times n$ matrix and $B$ is an $n \times p$ matrix. Then $AB$ is an $m \times p$ matrix. The $(i,j)$ entry of $AB$ is the $i$th row of $A$ times the $j$th column of $B$.

Clicker question

$ \left[ \begin{array}{cc} 0 & 0 \\ 0 & 1 \end{array} \right] \left[ \begin{array}{cc} 1 & 2 \\ 3 & 4 \end{array} \right] =$

  • A $\left[ \begin{array}{cc} 0 & 0 \\ 0 & 4 \end{array} \right]$
  • B $\left[ \begin{array}{cc} 0 & 0 \\ 3 & 4 \end{array} \right]$
  • C $\left[ \begin{array}{cc} 0 & 2 \\ 0 & 4 \end{array} \right]$

Other ways to look at it

The $i$th column of $AB$ is $A$ times (the $i$th column of $B$).

The $i$th row of $AB$ is (the $i$th row of $A$) times $B$.

The columns of $AB$ are linear combinations of the columns of $A$.

The rows of $AB$ are linear combinations of the rows of $B$.

Why?

Why is matrix multiplication defined the way it is?

One answer: it matches with the rule for matrix-times-vector, and it is associative.

Another answer: it turned out to be useful.

A lot of computer time is spent multiplying matrices. The Coppersmith-Winograd algorithm was the fastest from 1990 to 2010. Monday's SMBC comic refers to recent improvements.

Warning

Some things are true for multiplying numbers but not for multiplying matrices.

  • AB = BA
  • If AB = AC then B = C
  • If BA = CA then B = C
  • If AB = 0 then A = 0 or B = 0

Clicker question

$\left[ \begin{array}{cc} 1 & -1 \\ 1 & -1 \end{array} \right]^2$ is

  • A undefined
  • B $0_{2,2}$
  • C other

The transpose of a matrix

If $A$ is an $m \times n$ matrix then $A^T$ is an $n \times m$ matrix. The $i$th column of $A^T$ is the $i$th row of $A$.

The $i$th row of $A^T$ is the $i$th column of $A$.

The $(i,j)$ entry of $A^T$ is the $(j,i)$ entry of $A$.

Basic properties of addition and scalar multiplication

  • $A + B = B + A$
  • $(A + B) + C = A + (B + C)$
  • $A + 0 = A$
  • $r(A + B) = rA + rB$
  • $(r + s)A = rA + sA$
  • $r(sA) = (rs)A$

Matrix multiplication

  • $A(BC) = (AB)C$
  • $A(B + C) = AB + AC$
  • $(B + C)A = BA + CA$
  • $r(AB) = (rA)B = A(rB)$
  • $I_mA = A = AI_n$

Transpose

  • $(A^T)^T = A$
  • $(A + B)^T = A^T + B^T$
  • $(rA)^T = rA^T$
  • $(AB)^T = B^T A^T$
