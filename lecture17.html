§4.5: The dimension of a vector space

TL;DR The dimension of a vector space is the number of vectors in a basis.

Correction to Theorem 8

I forgot "onto" in the statement of Theorem 8.

If $\mathcal{B} = \{ \mathbf{b}_1, \dots, \mathbf{b}_n \}$ is a basis for a vector space $V$ then $$\mathbf{v} \mapsto [\mathbf{v}]_\mathcal{B}$$ is a one-to-one and onto linear transformation from $V$ onto $\mathbb{R}^n$.

If you don't know what "one-to-one" and "onto" mean, look up "bijection".

A linear transformation $T$ is called one-to-one and onto if $T(\mathbf{x}) = \mathbf{b}$ always has exactly one solution.

If $T(\mathbf{x}) = \mathbf{b}$ sometimes has infinity solutions then $T$ is many-to-one.

If $T(\mathbf{x}) = \mathbf{b}$ has no solution for some $\mathbf{b}$ then $T$ did not map onto that $\mathbf{b}$.

Theorem 8 is more conceptual than useful to you at this point.

If $V$ was an abstract vector space, not columns of numbers, you might pick a basis and use coordinate vectors. Think of $\mathbf{v}$ as the "real" vector, and $[\mathbf{v}]_\mathcal{B}$ as "representing" $\mathbf{v}$. Theorem 8 says that no information is lost in this representation.

Confusion: change of coordinates

The change of coordinates matrix only applies when $V = \mathbb{R}^n$. Vectors are already columns of numbers, but for some reason you might prefer to represent them using a non-standard basis $\mathcal{B}$.

The map $\mathbf{v} \mapsto [\mathbf{v}]_\mathcal{B}$ is a linear map from $\mathbb{R}^n$ to itself, so it is given by an $n \times n$ matrix.

That matrix is $P_\mathcal{B}$, whose columns are the basis vectors. For a square matrix, "one-to-one and onto" is the same as "invertible".

$\mathbf{v} = P_\mathcal{B} [\mathbf{v}]_\mathcal{B}, \quad [\mathbf{v}]_\mathcal{B} = P_\mathcal{B}^{-1} \mathbf{v}$

This is more conceptual than useful to you at this point.

Theorem 9

If $V$ has a basis with $n$ vectors, then any set of more than $n$ vectors in $V$ must be linearly dependent.

Theorem 10

Any two bases for $V$ must have the same number of vectors.

[Added later: "bases" is the plural of "basis".]

Definition

If $\{ \mathbf{b}_1, \mathbf{b}_2, \dots, \mathbf{b}_n \}$ is a basis for $V$ then $n$ is the dimension of $V$.

Notation: dim$(V) = n$.

Zero: If $V = \{ \mathbf{0} \}$ then dim$(V) = 0$.

The set of all functions from $\mathbb{R}$ to $\mathbb{R}$ is a vector space that does not have a finite basis. It is infinite dimensional.

A subspace of $\mathbb{R}^n$ always has a finite basis. It is finite dimensional.

Theorem 11

If $H$ is a subspace of $V$ then $\operatorname{dim} H \le \operatorname{dim} V$.

Theorem 12

Let $V$ be a vector space and $\mathcal{B} = \{ \mathbf{b}_1, \mathbf{b}_2, \dots, \mathbf{b}_n\}$ be a set of $n$ different vectors in $V$.

Any two of the following implies the third:
  • $\mathcal{B}$ is linearly independent.
  • $\mathcal{B}$ spans $V$.
  • dim$(V) = n$.
